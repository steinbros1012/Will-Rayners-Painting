# Will Rayners Custom Painting

Marketing site for Will Rayners Custom Painting, built with Astro, React, and Tailwind CSS and deployed on Vercel.

## Scripts

- `npm run dev` starts the local dev server
- `npm run build` creates the production build
- `npm run preview` previews the production build
- `npm run check` runs Astro's project checks
- `npm run test:run` runs the Vitest suite

## Environment Variables

The contact and estimate email routes expect these variables:

- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_SECURE`
- `SMTP_USER`
- `SMTP_PASSWORD`
- `SMTP_FROM_EMAIL`
